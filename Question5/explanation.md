**Explanation:** Created an empty Markdown file to document the commands used and their explanations.

### Command:
```bash
touch answer.md
```
**Explanation:** Created a Markdown file to write the answer for the theory question.

```bash

```
**Explanation:** Opened the answer file in the Nano text editor to write the solution.

```bash

```
**Explanation:** Displayed the contents of the answer file to verify that it was saved correctly.
